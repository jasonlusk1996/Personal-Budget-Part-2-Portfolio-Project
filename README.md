# Personal-Budget-Part-1-Portfolio-Project

## Description

This is the second portfolio project for the CodeCademy back-end developer career course. This project is intended to manage the budget of different expenses, with each expense having a related envelope. This project contains an array of envelopes (objects) that each have a name, id, and budget associated. Using JavaScript functions and express routes, this project runs a server and allows the user to retrieve and change the information relating to an envelope. Examples of requests that users can make is listed below.

+ Get information about all envelopes
+ Get information about a specfic envelope based on id
+ Add a new envelope with a name and budget (id auto assigned)
+ Change the budget of an envelope based on id
+ Swap budgets between two envelopes
+ Apply the same budget to all envelopes
+ Withdraw from the budget of an envelope
+ Delete an enevelope based on id

## Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Language:** JavaScript

## Getting Started

1. Clone the repository: `git clone https://github.com/jasonlusk1996/Personal-Budget-Part-1-Portfolio-Project.git`
2. Install dependencies: `npm install`
3. Start the server: `node server.js`
